** Двоичное представление
typedef struct { int bits[4]; } s21_decimal; 
typedef struct { int bits[7]; } s21_big;  (6 значащих int и последний как в decimal знак с экспонентой)
** При обращении в функциях называю число целиком decimal, (s21_decimal decimal)
  отдельный unsigned int в массиве bits - number, (uint32_t number)
  позиция отдельного бита - position. 
  например s21_bit_get(uint32_t number, int position)
** Операции:
& логическое "и" (and)
| логическое "или" (or)
^ исключающее "или" (xor)
<< сдвиг влево
>> сдвиг вправо
~ инвертировать (из 1101 сделается 0010)
